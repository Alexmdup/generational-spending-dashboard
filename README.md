# generational-spending-dashboard
## Project Overview
PowerBI dashboard examining expenditure data across different generations utilizing data from the United Sates Burea of Labor Statistics.

## Tools Used
SQL, PowerBI, PowerQuery, DAX

## Dashboard Features
- Breakdown of average annual expenditures into major categories
- Comparison between total amount spent and percentage of income spent
- In-depth look at the three largest categories of spending (housing, transportation, and food)
